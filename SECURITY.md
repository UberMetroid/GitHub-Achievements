# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you find a security vulnerability, please open an issue with:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

**Do NOT** create a public issue for security vulnerabilities.

## This Project's Scope

This tool:
- Uses GitHub's official CLI (`gh`) for API calls
- Stores config locally in `~/.config/github-achievements/`
- Does NOT collect or transmit any personal data
- Does NOT require any sensitive permissions beyond what's needed for GitHub API

## Best Practices When Using This Tool

- Review the code before running
- Only grant necessary GitHub permissions
- Keep your GitHub CLI authenticated securely
