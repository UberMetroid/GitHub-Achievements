# GitHub Achievements – Earnable Badges

Below are the *current* earnable achievements and legit ways to pursue them.

| Badge | How to Earn | Tiers / Thresholds |
|---|---|---|
| Pull Shark | Open PRs that are **merged** | 2 / 16 / 128 / 1024 PRs |
| Galaxy Brain | Have Q&A answers accepted | 2 / 8 / 16 / 32 answers |
| Starstruck | Own a repo with stars | 16 / 128 / 512 / 4096 stars |
| Pair Extraordinaire | Co-author commits on merged PRs | 1 / 10 / 24 / 48 PRs |
| Quickdraw | Close an issue or PR within 5 minutes of opening | One-time |
| YOLO | Merge a PR without a code review | One-time |
| Public Sponsor | Sponsor a project via GitHub Sponsors | One-time |

## Legit Tactics (No Spam)

### Pull Shark
- Contribute to active repos with clear maintainer flow.
- Pick tasks from issues, open PRs, and respond to review.

### Galaxy Brain
- Answer real questions in **Discussions (Q&A)**.
- Focus on helpful, reproducible answers.

### Starstruck
- Build a repo that solves a real problem and promote it honestly.

### Pair Extraordinaire
- Co-author a commit in a PR **with explicit consent**.

### Quickdraw
- Open a small issue + close it with a fix quickly.

### YOLO
- Merge a PR without review (only if your repo policy allows it).

### Public Sponsor
- Sponsor a project via GitHub Sponsors.

## Guardrails

- No fake PRs, no meaningless spam, no reviewer bypass in other people’s repos.
- Use this repo for planning + tracking, not for gaming the system.
