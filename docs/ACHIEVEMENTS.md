## Legit Tactics (No Spam)

### Pull Shark
- Contribute to active repos with clear maintainer flow.
- Pick tasks from issues, open PRs, and respond to review.
- Look for "good first issue" labels.

### Galaxy Brain
- Answer real questions in **Discussions (Q&A)** section.
- Focus on helpful, reproducible answers.
- Browse repos with unanswered Q&A discussions.

### Starstruck
- Build a repo that solves a real problem.
- Write good documentation.
- Promote honestly on social media, Reddit, etc.

### Pair Extraordinaire
- Co-author a commit in a PR **with explicit consent**.
- Use `Co-authored-by: name <email>` in commit message.

### Quickdraw
- Open a small issue + close it with a fix quickly.
- Best done in your own repos first.

### YOLO
- Merge a PR without review (only if your repo policy allows it).
- Best for small fixes in your own repos.

### Public Sponsor
- Sponsor a project via [GitHub Sponsors](https://github.com/sponsors).

### Arctic Code Vault
- This was a one-time event in 2020.
- Contribute to any public repo before end of 2020.

### Llama (1,000 Contributions)
- Make contributions consistently throughout the year.
- Includes: commits, PRs, issues, reviews, etc.

### Hacker / Founder
- Simply create a repository (public for Hacker, any for Founder).

### Developer
- Upload a profile picture to your GitHub account.

## Guardrails

- No fake PRs, no meaningless spam, no reviewer bypass in other people's repos.
- Use this repo for planning + tracking, not for gaming the system.
- All achievements should be earned through genuine contributions.
